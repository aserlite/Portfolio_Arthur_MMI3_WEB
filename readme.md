
# Portfolio Arthur CUVILLON

Portfolio réalisé avec vite.js, tailwindCSS et Alpine.js dans le cadre d'un cours de BUT MMI 3eme année

## Installation

Installer mon projet avec npm (après avoir cloné)

```bash
  npm install 
  npm run dev
```
